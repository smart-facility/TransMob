Choose the content from "Activity Location" Table:

Apart from "Activity Location" file, there are another six files can be customised for Transport data.
Please refer to the "User Interface Manual" file for the copying or modifying instructions.
1. "Vehicle_Type": The Vehicle_Type file contains details of the vehicles used by TRANSIMS, for example, the length, maximum speed and acceleration and deceleration characteristics of each vehicle type.
2. "Transit_Schedule": The Transit_Schedule file contains information on each transit route including the transit departure time for each stop along the route. The values in the Transit_Schedule file are listed in chronological order along each route.
3. "Transit_Route": The Transit_Route file contains information about the route paths of transit vehicles in the network such as route ID, route name, transit mode, etc.
4. "Transit_Stop": The Transit_Stop file contains information about all transit stops that define locations on links where a transit vehicle, such as a bus or light rail, waits to embark and disembark passengers.
5. "Link": The Link file contains information that specifies the characteristics of a link. A link represents a segment of a transport network (e.g. a road) and is defined as a connection between two nodes (A and B).
6. "Signalized_Node_2": The Signalized_Node_2 file contains information about the type of controller at the nodes that are signalized.
